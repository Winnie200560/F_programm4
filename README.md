## Задание 1.
### Текст задачи
Дерево содержит строки. В конец каждой строки приписать ее же.
### Алгоритм решения
1. Пользователь вводит количество элементов.
2. Генерируем список случайных строк.
3. Строим бинарное дерево поиска с помощью функции вставки insert.
4. Создаем новое дерево с mapTree, в котором в каждой строке в конец приписана она сама.
5. Вывод нового дерева.

## Задание 2.
### Текст задачи
Выяснить, содержится ли в дереве элемент с заданным значением.
### Алгоритм решения
1. Пользователь вводит количество элементов.
2. Генерируем список случаных чисел в диапазоне (1,100).
3. Строим бинарное дерево поиска через фунюцию insert.
4. Пользователь вводит число для поиска в дереве.
5. Обходим дерево рекурсивно через fold и проверяем наличие элемента.
6. Для поиска элемента используется функция includeValue, которая проверяет равенство текущего элемента с заданным значением.
7. Возврщаем true или false в зависимости от поиска результата.
