## Задание 1.
### Текст задачи
Дерево содержит строки. В конец каждой строки приписать ее же.
### Алгоритм решения
1. Пользователь вводит глубину дерева n.
2. Рекурсивная функция createTree создает бинарное дерево глубины n, заполняя его случайными строками.
3. Функция printTree выполняет симметричный обход дерева и выводит его.
4. Функция mapTree рекурсивно проходит по всем узлам и применяет функцию удваивания строки к каждому элементу.
5. Результат - новое дерево, где каждая строка удвоена.

## Задание 2.
### Текст задачи
Выяснить, содержится ли в дереве элемент с заданным значением.
### Алгоритм решения
1. Пользователь вводит глубину дерева n.
2. Рекурсивная функция createTree создает бинарное дерево, заданной глубины, состоящее из чисел в диапазоне (1,100).
3. Пользователь вводит число для поиска.
4. Функция printTree выполняет симметричный обход дерева и выводит значения дерева.
5. Функция foldTree рекурсивно проходит по всем узлам дерева и накапливает результат.
6. Для поиска элемента используется функция includeValue, которая проверяет равенство текущего элемента с заданным значением.
8. Если элемент найден, результат - true и выводится соотвествующее сообщение.
